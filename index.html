Google Drive sync

🔗
Connect Google Drive
Auto-save your data to a file in Drive

You need a free Google API Client ID. It takes about 2 minutes to set up. Follow these steps:

1
Go to console.cloud.google.com and create a new project (name it anything, e.g. "Float App").
2
In the left menu go to APIs & Services → Library. Search for Google Drive API and click Enable.
3
Go to APIs & Services → OAuth consent screen. Choose External, fill in an app name (e.g. "Float"), your email, and save.
4
Go to Credentials → + Create Credentials → OAuth Client ID. Choose Web application. Under Authorised JavaScript origins add the URL where you open this file (or null if opening locally). Copy the Client ID.
5
Paste your Client ID below and tap Connect.
